## 📖 Sobre o Projeto

O **MotoGear** é um projeto front-end de uma loja virtual voltada para equipamentos e acessórios de motociclismo. 

Este projeto foi desenvolvido como atividade somativa para a disciplina de Fundamentos de Programação Web do curso de Análise e Desenvolvimento de Sistemas (PUCPR). O objetivo principal foi aplicar os conceitos essenciais de estruturação, estilização e interatividade web utilizando tecnologias nativas (Vanilla), sem o auxílio de frameworks ou bibliotecas externas.

🔗 **[Acesse o projeto rodando ao vivo aqui](https://nicolasm3deiros.github.io/projeto-motorgear/)**

---

## 🎯 Funcionalidades e Requisitos Atendidos

* **Interface Responsiva:** Layout adaptável para dispositivos móveis e desktops utilizando `CSS Grid` e `Media Queries`, além do uso adequado da meta tag `viewport`.
* **Organização Estrutural:** O projeto conta com 4 páginas conectadas (`index.html`, `form.html`, `formAction.html`, `about.html`).
* **Tratamento de Imagens:** Utilização de propriedades como `object-fit: contain` para padronização de imagens sem distorções em um grid de 10 produtos.
* **Validação de Formulários (JavaScript):** Validação client-side para garantir o preenchimento correto dos dados de contato, incluindo o uso de Expressões Regulares (Regex).
* **Manipulação de Dados (Método GET):** Envio de dados do formulário via URL (GET) e captura das informações na página de destino (`formAction.html`) utilizando `URLSearchParams` para exibir uma mensagem personalizada ao usuário.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, design responsivo, CSS Grid e efeitos visuais (parallax, hover, transições).
* **JavaScript (Vanilla):** Lógica de programação, manipulação do DOM e validação de dados.
